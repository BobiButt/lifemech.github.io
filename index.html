Android Based Linux

1) Phoenix OS:

This Android OS is based on Android version 5 or 6 (based on the version you use). Mostly it is use for playing games. It provide a keymapper that help us to play games or use application with our keyboard or mouse.It runs smoothly on old PC.

2) Prime OS:

This Android OS is based on Android 7 and Android 12(based on different version).Prime OS is very popular as it gives the best keymapper as well as stability in running applications and games.It is divided in different categories i.e. Classic, Sandard, and Mainline. Classic version is for old PC or Laptop. Standard version is for PC that can run Windows Vista or Windows 7. Mainline version is for mordern pc or laptop.You can download the version that is suitable for your PC or Laptop.

3) Bliss OS:

Bliss OS is another Android OS that can run android apps and games.It avaliable in Android 11 and Android 12.It required CPU with Intel Core i3 processor.There is no any built-in keymapper but you can download it from browser or Google Playstore.It gives best stability of running apps.Most of the users recommended it.

4) Remix OS:

Remix OS is the best Android OS based on linux that can run on old Laptops or PC very smoothly. It gives a computer a new birth. You can run most of the apps and games from Google Playstore.Moreover it comes with builtin keymapper that helps the user to interact their keyboard with apps or games.There is no mouse mapping in this keymapper. So,you have to download another keymapper that suits you.It based on android 6.It is also known as Dead OS as there is no update of this OS anymore.It is very stable and chance of app crash is very low.

5) Android x86:

Android x86 is avaliable on almost all version of Android. You can download old version of Android x86 that suits to your old laptop or computer. Old PC or Laptop can run android 5 version or below it.It provide the best stability as well as performance that make your Laptop as mordern one.The one big disadvantage of this OS is that, it is lack of keymapper. You have to download it from browser or Google Playstore.

6) Fyde OS:

Fyde OS is also known as modded version of Chrome OS. The difference is that Chrome OS is avaliable for specific devices.But this OS is avaliable for almost all devices.It based on Android 9.It gives best performance in low end devices.if you have Intel Core i3 processor then you can install it.As Android x86 and Bliss OS, it also has no keymapper.You have to download it from Google Playstore or browser.

6) Chrome OS Flex:

Chrome OS Flex is the one the best OS for low end PC. It runs smoothly in any 10 to 20 years old laptop or pc. It is lack of android system.Its mean you cant downlaod android application on it.It provided the Chrome Web Store for downlaod apps.It is comes with builtin software like Google Sheet ,Youtube, Google Translator and many more so you can watch movies,and can done your office task very easily and with high performance. It is the best os I recommended for oldest computers that can run only Windows XP.
